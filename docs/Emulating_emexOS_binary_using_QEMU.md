# Running emexOS binary using QEMU

The easiest way to run emexOS is to run it in QEMU. 

- Install QEMU in your machine.
- Download the latest emexOS version from [github](https://github.com/emexos/emexOS1/releases).
- Open your Virtual Machine Manager and select "Create a new virtual machine" and then select "Local install media".
- Select "Browser" and find the the emexOS iso by pressing "Browse Local".
- Uncheck the "Automatically detect..." option and type "Generic Linux 2024".
- Assign at least 512MB of RAM and 2 cores.
- Choose either to enable storage or not to.
- Keep the default options and select "finish".
